STEP 1

Input Processing

User submits natural language text via web UI or API. Input is sanitized, validated, and rate-checked before any processing begins.

STEP 2

NLP Extraction

The fine-tuned NER model extracts structured entities: DRUG, DOSE, FREQUENCY, and TIME — converting unstructured text into actionable data.

STEP 3

Schedule & Persist

Reminder times are computed, stored in PostgreSQL, and indexed for fast scheduler lookups via GIN indexing on ARRAY columns.

STEP 4

Notify

APScheduler ticks every minute, matching current time against reminder arrays and firing Telegram push notifications to each patient.
